# Pomodoro GUI
Beat procrastination and improve your focus one pomodoro at a time
<br>
<br>
<br>
#### Clone the repo
`git clone https://github.com/shahan007/Pomodoro-GUI`

#### Set up the environment
`python -m venv venv`

`source venv/Scripts/activate`

`pip install -r requirements.txt`

#### Start the game
`python run.py`
